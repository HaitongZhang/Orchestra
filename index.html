Abstract:

Usually, text-to-speech and voice conversion is considered as two tasks and tackled by separate models. In this paper, we proposed a multi-modal voice cloning system called Orchestra, which aims to support text-to-speech (TTS) and voice conversion (VC) simultaneously. The system mainly contains three modules. The first module is vector-quantization Variational-Autoencoder (VQ-VAE), which aims to extract unsupervised linguistic units given audio. The second module is a Tacotron2-like sequence-to-sequence model, which generates mel-spectrogram given either unsupervised linguistic units or phonemes. The third module is a neural vocoder to generate waveform conditioned on mel-spectrogram. We evaluate our proposed system in both TTS and VC. In the TTS task, we found that the proposed system gives a comparable performance to Tacotron2 in the multi-speaker setting. When cloning new voices using 2-minute data, the proposed system outperforms Tacotron2 on almost all evaluation metrics. In the VC task, we found that the proposed system can significantly improve the conversion performance, compared with the VQ-VAE based VC system.
